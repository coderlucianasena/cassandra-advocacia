## 📋 Sobre o Projeto

Site institucional moderno e responsivo desenvolvido para o escritório de advocacia **Jucá Flexa Advogadas Associadas**. O projeto visa estabelecer uma presença online profissional, apresentar os serviços jurídicos de forma clara e facilitar o contato com clientes.

### 🎯 Objetivos
- ✅ Estabelecer presença online profissional
- ✅ Apresentar serviços jurídicos de forma clara
- ✅ Facilitar contato com clientes
- ✅ Compartilhar conhecimento através do blog
- ✅ Construir credibilidade e confiança

## 🚀 Como Executar

### **Pré-requisitos**
- Navegador web moderno
- Servidor web local (opcional)

### **Instalação**
```bash
# Clone o repositório
git clone https://github.com/coderlucianasena/cassandra-advocacia.git

# Navegue para o diretório
cd cassandra-advocacia

# Abra o arquivo index.html no navegador
# Ou use um servidor local:
python -m http.server 8000
# Acesse: http://localhost:8000
```
